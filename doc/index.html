Tutorials and workshops¶

  • The MapFish In Production Workshop

    Presented at FOSS4G 2011. This workshop shows the tools and practices the Camptocamp teams use for developing, deploying, and running MapFish applications. This workshop is a also a guide to MapFishSample, the official MapFish demo application.

  • The MapFish tutorial (Version française du tutorial)

    Presented at FOSS4G 2010. This tutorial provides an overview of the MapFish framework.

  • The Python Workshop (Version française du workshop)

    Presented at FOSS4G 2010. This workshop teaches the basics of Pylons, SQLAlchemy, GeoAlchemy, Shapely, geojson, and MapFish.
