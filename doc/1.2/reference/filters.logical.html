mapfish.lib.filters.logical¶

class mapfish.lib.filters.logical.Logical(type, filters=None)¶

Logical filter.

type
the type of filter to create. Possible values are Logical.NOT, Logical.AND or Logical.OR.
filters=None
a list of filters this filter will create logical expressions with. filters can also be added after the creation of the logical filter, using logical_filter.filters.append(filter).
to_sql_expr()¶
Return the SQLAlchemy SQL expression corresponding to that filter.
